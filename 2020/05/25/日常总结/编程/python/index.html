python中的变量创建

python 变量命名

在 Python 程序中，变量是用一个变量名表示，可以是任意数据类型，变量名必须是大小写英文、数字和下划线（_）的组合，且不能用数字开头

eg:

列表与元组

列表

eg:

list1=[“1”,2,”three”]

即列表名,’[]’ 列表符组成

列表中的元素类型可以不一致.

如何访问列表中的值

  1. 通过索引下标来访问

即已知索引长度为3

我们可以通过
list1[i]

i为0~2之间的整数

这与java中的数组类似

2.通过下标区间来访问

如 list1[0:2]
可以取到list[0],list[1]
这里是因为[0:2]是左闭右开的
